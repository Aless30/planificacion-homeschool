Bloque 1: Matematicas — Factorizacion (refuerzo con paso a paso)

Repaso rapido: los 3 tipos de factorizacion

La semana pasada vimos 4 tipos. Esta semana vamos a profundizar en los 3 mas importantes con ejercicios completamente resueltos para que veas el proceso de pensamiento completo.

Recuerda la idea general: Factorizar es tomar una expresion "armada" y descomponerla en partes mas simples multiplicadas entre si. Es como desarmar un mecanismo para ver sus piezas.

TIPO 1: Factor comun — Ejercicio resuelto

Factorizar: 8x³ - 12x²

Paso 1 — Miro cada termino por separado:

  • Primer termino: 8x³ → esta compuesto por 8 · x · x · x
  • Segundo termino: 12x² → esta compuesto por 12 · x · x

Paso 2 — Me pregunto: ¿que tienen en comun?

Miro los numeros:

  • 8 y 12. ¿Cual es el mayor numero que divide a ambos? Pruebo: ¿8? No, porque 12÷8 no es exacto. ¿4? Si: 8÷4=2 y 12÷4=3. El maximo comun divisor de los numeros es 4.

Miro las letras:

  • El primer termino tiene x³ (tres equis multiplicadas: x · x · x)
  • El segundo termino tiene x² (dos equis: x · x)
  • ¿Cuantas x comparten ambos? El minimo: x² (dos equis)

Paso 3 — Defino el factor comun:

Factor comun = 4x² (el 4 de los numeros y x² de las letras)

Paso 4 — Divido cada termino entre el factor comun:

  • 8x³ ÷ 4x² = 2x (porque 8÷4=2 y x³÷x²=x)
  • 12x² ÷ 4x² = 3 (porque 12÷4=3 y x²÷x²=1)

Paso 5 — Escribo el resultado:

8x³ - 12x² = 4x²(2x - 3)

Paso 6 — Verifico (multiplico de vuelta):

  • 4x² · 2x = 8x³ ✓
  • 4x² · (-3) = -12x² ✓

Suma: 8x³ - 12x² ✓ Correcto.

TIPO 2: Diferencia de cuadrados — Ejercicio resuelto

Factorizar: 4a² - 9b²

Paso 1 — ¿Es una diferencia de cuadrados? Verifico 3 condiciones:

  • ¿Son dos terminos? → Si: 4a² y 9b² ✓
  • ¿Estan restados? → Si: 4a² menos 9b² ✓
  • ¿Ambos son cuadrados perfectos? Necesito verificar:
    • 4a² = ¿(algo)²? → 4 = 2² y a² = (a)². Entonces 4a² = (2a)² ✓
    • 9b² = ¿(algo)²? → 9 = 3² y b² = (b)². Entonces 9b² = (3b)² ✓

Si, es una diferencia de cuadrados.

Paso 2 — Identifico que esta elevado al cuadrado en cada termino:

  • En 4a², lo que esta al cuadrado es 2a (porque (2a)² = 4a²)
  • En 9b², lo que esta al cuadrado es 3b (porque (3b)² = 9b²)

Paso 3 — Aplico la formula a² - b² = (a + b)(a - b):

4a² - 9b² = (2a)² - (3b)² = (2a + 3b)(2a - 3b)

Paso 4 — Verifico multiplicando de vuelta:

  • 2a · 2a = 4a²
  • 2a · (-3b) = -6ab
  • 3b · 2a = +6ab
  • 3b · (-3b) = -9b²

Sumo: 4a² - 6ab + 6ab - 9b² = 4a² - 9b² ✓ (los -6ab y +6ab se cancelaron)

TIPO 3: Trinomio x² + bx + c — Ejercicio resuelto

Factorizar: x² + 2x - 15

Paso 1 — Identifico b y c:

  • La expresion es: x² + 2x - 15
  • b = +2 (el numero que acompaña a x)
  • c = -15 (el numero solo, sin x)

Paso 2 — Deduzco los signos de los numeros que busco:

  • c es negativo (-15) → los dos numeros tienen signos diferentes (uno positivo, uno negativo). Esto es porque positivo × negativo = negativo.
  • b es positivo (+2) → al sumar los dos numeros, "gana" el positivo. Eso significa que el numero mas grande en valor absoluto es el positivo.

Busco: un positivo grande y un negativo mas chico, que multiplicados den -15 y sumados den +2.

Paso 3 — Hago una lista organizada de posibilidades:

¿Que pares de numeros con signos diferentes me dan -15 al multiplicarse?

Numero 1Numero 2¿Multiplican -15?¿Suman +2?
+1-15(+1)(-15) = -15 ✓1 + (-15) = -14 ✗
+15-1(+15)(-1) = -15 ✓15 + (-1) = +14 ✗
+3-5(+3)(-5) = -15 ✓3 + (-5) = -2 ✗
+5-3(+5)(-3) = -15 ✓5 + (-3) = +2 ✓

¡Los encontre! Los numeros son +5 y -3.

Paso 4 — Escribo la factorizacion:

x² + 2x - 15 = (x + 5)(x - 3)

Paso 5 — Verifico (multiplico de vuelta):

  • x · x = x²
  • x · (-3) = -3x
  • 5 · x = +5x
  • 5 · (-3) = -15

Sumo todo: x² - 3x + 5x - 15 = x² + 2x - 15 ✓ Correcto.

DIA 2 — MIERCOLES

Bloque 1: Matematicas — Ecuaciones cuadraticas (refuerzo con paso a paso)

Repaso: ¿que es una ecuacion cuadratica?

Es una ecuacion donde la variable aparece elevada al cuadrado: ax² + bx + c = 0

Hay dos metodos para resolverla. Hoy vamos a ver un ejercicio completo de cada uno, paso a paso.

Metodo 1: Resolver por factorizacion — Ejercicio resuelto

Resolver: x² + x - 12 = 0

Paso 1 — Reconozco que puedo factorizar el lado izquierdo.

Es un trinomio de la forma x² + bx + c, con b = +1 y c = -12.

Paso 2 — Factorizo (usando el metodo del lunes):

  • c es negativo (-12) → signos diferentes
  • b es positivo (+1) → el mas grande es el positivo
  • Busco dos numeros que multiplicados den -12 y sumados den +1:
Num 1Num 2¿Multiplican -12?¿Suman +1?
+2-6+2 × -6 = -12 ✓2 + (-6) = -4 ✗
+3-4+3 × -4 = -12 ✓3 + (-4) = -1 ✗
+4-3+4 × -3 = -12 ✓4 + (-3) = +1 ✓

Factorizacion: x² + x - 12 = (x + 4)(x - 3)

Paso 3 — Aplico el principio del producto cero:

Si (x + 4)(x - 3) = 0, entonces uno de los dos factores debe ser cero:

  • x + 4 = 0 → x = -4
  • x - 3 = 0 → x = 3

Paso 4 — Verifico ambas soluciones en la ecuacion original:

  • x = -4: (-4)² + (-4) - 12 = 16 - 4 - 12 = 0 ✓
  • x = 3: (3)² + (3) - 12 = 9 + 3 - 12 = 0 ✓

Ambas soluciones son correctas.

Metodo 2: Formula general — Ejercicio resuelto

Resolver: 2x² + 5x - 3 = 0

Paso 1 — Identifico a, b y c:

  • a = 2 (el numero que acompaña a x²)
  • b = 5 (el numero que acompaña a x)
  • c = -3 (el numero solo)

Paso 2 — Calculo el discriminante (Δ = b² - 4ac):

  • Δ = (5)² - 4(2)(-3)
  • Δ = 25 - (-24)
  • Δ = 25 + 24 = 49

El discriminante es 49 (positivo), asi que hay dos soluciones reales diferentes.

Paso 3 — Aplico la formula x = (-b ± √Δ) / 2a:

  • x = (-5 ± √49) / (2 × 2)
  • x = (-5 ± 7) / 4

Paso 4 — Calculo las dos soluciones:

  • x₁ = (-5 + 7) / 4 = 2/4 = 1/2
  • x₂ = (-5 - 7) / 4 = -12/4 = -3

Paso 5 — Verifico ambas:

  • x = 1/2: 2(1/2)² + 5(1/2) - 3 = 2(1/4) + 5/2 - 3 = 1/2 + 5/2 - 3 = 6/2 - 3 = 3 - 3 = 0 ✓
  • x = -3: 2(-3)² + 5(-3) - 3 = 2(9) - 15 - 3 = 18 - 15 - 3 = 0 ✓

DIA 3 — VIERNES

Bloque 1: Matematicas — Sistemas de ecuaciones (refuerzo con paso a paso)

Repaso: ¿que es un sistema de ecuaciones?

Son dos ecuaciones que deben cumplirse al mismo tiempo con las mismas incognitas. Es como un acertijo con dos pistas que juntas te dan la respuesta.

Metodo de sustitucion — Ejercicio resuelto

Resolver: x + y = 10 y 3x + 2y = 26

Paso 1 — Elijo una ecuacion y despejo una variable:

De la primera ecuacion (es la mas simple), despejo x:

x + y = 10 → x = 10 - y

Paso 2 — Sustituyo esa expresion en la OTRA ecuacion:

En la segunda ecuacion, donde dice "x", pongo "(10 - y)":

3(10 - y) + 2y = 26

Paso 3 — Resuelvo la ecuacion resultante (solo tiene una variable):

  • 30 - 3y + 2y = 26 (distribuyo el 3)
  • 30 - y = 26 (simplifico: -3y + 2y = -y)
  • -y = 26 - 30
  • -y = -4
  • y = 4

Paso 4 — Encuentro la otra variable sustituyendo de vuelta:

x = 10 - y = 10 - 4 = 6

Paso 5 — Verifico en AMBAS ecuaciones originales:

  • Primera: x + y = 6 + 4 = 10 ✓
  • Segunda: 3(6) + 2(4) = 18 + 8 = 26 ✓

Solucion: x = 6, y = 4.

Metodo de eliminacion — Ejercicio resuelto

Resolver: 5x - 3y = 1 y 2x + 3y = 13

Paso 1 — Observo si alguna variable se puede eliminar directamente:

Miro los coeficientes de "y": en la primera es -3y y en la segunda es +3y. ¡Son opuestos! Si las sumo, se cancelan.

Paso 2 — Sumo ambas ecuaciones:

  • (5x - 3y) + (2x + 3y) = 1 + 13
  • 5x + 2x - 3y + 3y = 14
  • 7x = 14 (los -3y y +3y se cancelaron)
  • x = 2

Paso 3 — Sustituyo x en cualquiera de las ecuaciones originales:

Uso la segunda: 2(2) + 3y = 13 → 4 + 3y = 13 → 3y = 9 → y = 3

Paso 4 — Verifico en la OTRA ecuacion (la que no use en el paso 3):

Primera: 5(2) - 3(3) = 10 - 9 = 1 ✓

Solucion: x = 2, y = 3.

Ahora tu — Sistemas de ecuaciones

  1. x + y = 12 y x - y = 4 (pista: si las sumas, se eliminan las y)
  2. 2x + y = 7 y x - y = 2
  3. 3x + 2y = 12 y x + 2y = 8 (pista: si las restas, se eliminan las 2y)
